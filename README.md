# pythonFlaskAPI

## Getting Started

1. install the packages `pip3 install -r requirements.txt`
2. uncomment line 23 in `main.py`
   1.  `db.create_all()`
3. start the server `python3 main.py`
4. recomment line 23 in `main.py`
   1. `db.create_all()`
5. save main.py
6. run the test `python3 test.py`
